| คำศัพท์ | **backup set** |
|---------|-------------------|
| **Glossary** | คำที่ใช้อธิบายกลุ่มของไฟล์หรือโฟลเดอร์บนไดรฟ์เดียวจากการสำรองข้อมูลครั้งเดียว ส่วนกลุ่มของเทปเรียกว่า family set |
| **ChatGPT** | "ชุดข้อมูลสำรอง" หมายถึง กลุ่มไฟล์หรือโฟลเดอร์ที่ถูกสำรองจากไดรฟ์เดียวกันในการสำรองข้อมูลครั้งเดียว ส่วนถ้าเป็นกลุ่มเทปหลายม้วนจะเรียกว่า "family set" |
| **Claude** | กลุ่มของไฟล์หรือโฟลเดอร์ที่อยู่บนพื้นที่จัดเก็บข้อมูล (volume) เดียวกันที่ได้จากการสำรองข้อมูลครั้งเดียว ในขณะที่กลุ่มของเทปสำรองหลายม้วนจะเรียกว่า family set |
| **ความหมายที่ตัวเราเข้าใจ** | ชุดไฟล์สำรองจากการทำ backup ครั้งเดียว |
